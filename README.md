# מסע בארץ ישראל – גרסה סטטית

הגרסה הזאת אינה דורשת React, Vite, npm או שרת פיתוח.

## פתיחה מקומית

פתחו ישירות את הקובץ:

`index.html`

## העלאה ל־GitHub Pages

העלו את כל הקבצים והתיקיות אל שורש ה־Repository:

- `index.html`
- `css`
- `js`
- `assets`

לאחר מכן הפעילו GitHub Pages מתוך ענף `main` ותיקיית `/root`.

## מה תוקן

הגרסה הקודמת הייתה פרויקט מקור של React + Vite ולכן פתיחת `index.html`
ישירות או העלאת קובצי המקור ללא תהליך Build יכלה לגרום לשגיאת 404.

הגרסה הזאת משתמשת רק ב־HTML, CSS ו־JavaScript רגילים,
וכל הנתיבים בה יחסיים.
