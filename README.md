# מסע בארץ ישראל – מעטפת ראשונית

פרויקט React + Vite עבור משחק למידה אינטראקטיבי ברוח העיצוב של רשת אמית.

## מה כלול בגרסה הזאת

- מסך בית מלא.
- לוגו רשת אמית בפינה הימנית העליונה.
- צבעי המותג שסופקו.
- אזור פתיחה עם איור מסע מקורי.
- כפתור התחלת מסע.
- חמש תחנות בעיצוב אחיד.
- מצב תחנה פתוחה/נעולה.
- חלון זמני לתחנה הראשונה.
- התאמה למחשב, טאבלט וטלפון.
- רכיב `StyledCircularIcon` עם fallback ל־`lucide-react`.

## הפעלה מקומית

```bash
npm install
npm run dev
```

## בנייה ל־GitHub Pages

```bash
npm run build
```

הקבצים המוכנים ייווצרו בתיקיית `dist`.

## הערה לגבי הפונט Simpler

הפרויקט מגדיר את `Simpler` כגופן ראשון ב־CSS, אך קובץ הגופן עצמו אינו כלול.  
כאשר קובץ מורשה יהיה ברשותך, ניתן להוסיף אותו מקומית ולהגדיר `@font-face`.

## השלב הבא

לאחר אישור הנראות הכללית, מומלץ לבנות את תחנת „נחלי ישראל” כתחנה מלאה ראשונה.
